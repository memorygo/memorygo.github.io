搜索

在 Vim 中寻找特定字或字符串，你可以使用 Vim 的搜索功能。有几种方式可以实现这一点，但最基本和最常用的方法是使用 / 命令。下面是一些基本步骤和技巧：

1. 使用 / 命令搜索

  • 开始搜索：按下 / 键后，Vim 会进入搜索模式

  • 输入搜索字符串：在 / 后面输入你的搜索词，然后按 Enter 键。Vim 会高亮显示第一个匹配项。

  • 跳转到下一个匹配项：按 n 键可以跳转到下一个匹配项。

  • 跳转到上一个匹配项：按 N 键（注意是大写的 N）可以跳转到上一个匹配项。

2. 使用 ? 命令反向搜索

  • 开始反向搜索：与 / 命令类似，当你想要从当前位置反向搜索（即向上搜索），可以使用 ? 命令。

  • 跳转到下一个/上一个匹配项：与 / 命令方向相反。即 n 向上搜索，N 向下搜索。

3. 搜索技巧

  • 大小写敏感：默认情况下，Vim 的搜索是区分大小写的。但你可以通过 :set ignorecase 命令来设置搜索不区分大小写，或使用 :set noignorecase 命令来恢复区分大小写的搜索。

  • 精确匹配：如果你想要精确匹配某个单词或字符串（包括大小写），可以在搜索字符串前加上 \< 和在字符串后加上 \>。例如，/\<word\> 会匹配单词 “word”，但不会匹配 “words” 或 “wordy”。

  • 正则表达式：Vim 的搜索功能支持强大的正则表达式。例如，使用 .* 可以匹配任意数量的任意字符。

4. 重复上次搜索

  • 如果你想要重复上次的搜索（无论方向），可以简单地按 n 或 N，无需再次输入搜索字符串。
