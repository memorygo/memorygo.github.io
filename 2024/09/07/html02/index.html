html5基础二

常用标签 2

###表格

  • 用于展示数据

标签

  • <table> </table> 用于定义表格的标签

  • <tr> </tr> 用于定义表格中的行

  • <td> </td> 用于定义表格行中的单元格

  • <th></th> 表头单元格，居中加粗，一般在表格首行

表格属性

不常用，直接通过css设置

属性直接写在table标签里
<table align="center"></table>

属性名 属性值 描述
align left、center、right、 对齐
border 1或”” 1:有边框，””:无边框
cellpadding 像素值 单元格边缘和内容间的留白，默认为1px
cellspacing 像素值 单元格之间的留白,默认为2px
width 像素值或百分比 表格的宽度
height 像素值或百分比 表格的高度

表格结构

<thead></thead> 表格头部区域
<tbody></tbody> 表格主体区域
作用:使语义层次分明，结构清晰

合并单元格

  • 方式：

跨行合并:rowspan="合并个数"
跨列合并: colspan="合并个数"

目标单元格

跨行:上行单元格
跨列:左列单元格
记得删除多余的单元格

列表

用于布局

  • 无序列表（重点）

<ul> </ul>
ul内只能放li标签,父标签
<li> </li>
li内可以随便放标签

无序列表有自己的样式属性，但常用css设置样式

有序列表

​<ol> </ol>
​<li> </li>

自定义列表(重点)

​<dl> </dl> 定义列表
​<dt>名词</dt>
​<dd> </dd> 名词解释

表单

  • 用于收集用户信息

组成：

  1. 表单域
  2. 表单控件（表单元素）
  3. 提示信息

表单域

​<form>用于定义表单域，以实现用户信息的收集和传递
​<form> 会把它范围内的表单元素信息提交给服务器.

常用属性

<form action="url地址" method="提交方式" name="表单域名称"> 表单控件元素 </form>

属性 属性值 作用
action url地址 用于指定接受表单数据的服务器程序的url
method get/post 设置提交方式
name 名称 用于指定表单名称,区分同页面多个表单域
  • 表单控件
  • input输入表单元素
  • select下拉表单元素
  • textarea文本域元素

<input> 表单元素

  • 作用：用于收集用户信息

<input type=""/> 单标签 根据type的属性，输入字段可以为多种形式（按钮，单选按钮，复选框，文本字段，掩码后的文本控件等）

  • type设置不同的属性值用来指定不同控件类型

type属性

属性值 描述
button 定义 可点击按钮（多数情况下，通过js启动脚本）
checkbox 复选框
file 输入字段和”浏览”按钮，供文件上传
hidden 隐藏的输入字段
image 图像形式的提交按钮
password 密码字段，其字符被掩码
radio 单选按钮
reset 重置按钮，清除表单中所有数据
submit 提交按钮，把表单域内的表单数据发送到服务器
text 单行输入字段，文本，默认20个字符宽

其它属性

属性 属性值 描述
name 自定义 定义input的名称
value 自定义 规定input的值
checked checked 规定此input元素首次加载时应当被选中
maxlength 正整数 输入字段中最大字符长度

name 和 value 是每个表单元素都要有的属性值，主要给后台人员用

要求单选按钮和复选框要有相同的name值
