css基础2

css三种样式表

  • 行内样式表（行内式）
  • 内部样式表（嵌入式）
  • 外部样式表（链接式）

  1. 内部样式表
    所有css代码单独放在一个<style>标签中

    <style>标签可以放在html文档任意地方，但一般放在<head>标签内

  2. 行内样式表
    在标签内部的style属性中设置css。适用于简单样式修改
    <div style="color: red; font-size: 12px;">xxx<div>

    要用双引号，写法符合css规范

  3. 外部样式表

    • 样式单独写在css文件，用时再引入

      css文件内不用写script标签

    • 用<link>标签引入css文件
      <link rel="stylesheet" href="css文件路径">
样式表 范围
行内样式表 控制一个标签
内部样式表 控制一个页面
外部样式表 控制多个页面

css复合选择器

  • 更高效、准确的选择目标元素
  • 由两个及以上基础选择器、通过不同方式组成
  • 常用包括: 后代选择器、子选择器、并集选择器、伪类选择器等
  1. 后代选择器（重要）
父代元素 后代元素 后代元素 ... {
    color: red;
    ...
} 

>选择改变的是最后代标签
元素可以用id选择器和类选择器
  1. 子选择器（重要）
父元素>子元素 {
    color: red;
    ...
}
>只能选择父代的直接子代
  1. 并集选择器（重要）
元素1,元素2 {
    color: red;
    ...
}
>元素可以为任何选择器
  1. 伪类选择器
向某些选择器添加特殊效果
  • 链接伪类选择器
a:link      <!-- 选择所有未被访问过的链接 -->
a:visited   <!-- 选择所有已被访问过的链接 -->
a:hover     <!-- 选择鼠标指针位于其上的链接 -->
a:active    <!-- 选择活动链接（鼠标正在按下还未抬起的链接）-->

lvha 写的先后顺序不能倒
页面中a标签需要单独设计样式（用链接伪类选择器）
  • focus 伪类选择器
    用于选取获得焦点的表单元素
input:focus {
    background-color: red;
}
>焦点就是光标，一般在input类元素上才能获取

css元素显示样式

  • 标签以什么方式进行显示
    html 元素一般分为块元素和行内元素

  • 块级元素
    常见块级元素有<h1>、<p>、<div>、<ul>、<li>等
    特点:
    a. 独占一行
    b. 宽高、内外边距可控

- 文字类块级标签如<h1>,<p>内部不能放块级元素

-行内元素(内联元素)
常见有<a>,<strong>,<b>,<em>,<i>,<del>,<s>,<ins>,<u>,<span>

特点:
a. 高宽直接设置无效
b. 默认宽度为自身内容的宽度
c. 行内元素内放不了块级元素

特殊

<a>标签可以放块级元素

  • 行内块元素
    特殊的行内元素，如<img/>,<input/>,<td>,同时具有块元素和行内元素的特点

    特点：
    a. 一行可以放多个
    b. 高度,行高，内外边距可以控制

    将a标签设置为块级元素

    将a标签设置为行内元素

css的背景

background-color --->背景颜色   默认为 transparent 透明
background-image --->背景图片   url地址、none
background-repeat --->背景图像的平铺方式    repeat、no-repeat、repeat-x、repeat-y
background-position --->背景图片的位置

background-position

background-position:x y;

>参数是 x 和 y 坐标 , 可以使用方位名词或精确单位
方位名词：top | center | bottom | left | center | right
精确单位：x坐标 | y坐标 （px）
