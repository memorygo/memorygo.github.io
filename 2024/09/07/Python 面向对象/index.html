Python 面向对象

基本概念

面向对象是一种编程范式，在此理念中，程序中的数据 和操作数据的函数是一个逻辑上的整体，我们称之为 对象，我们可以通过创建对象、向对象发出信息、对象间协作 来解决问题。

基本骨架

定义类

类是对象的模板，我们可以对其实例化得到一个对象。

  • 可以说 类是抽象的概念，对象是具体的东西

实例化

类通过实例化可以变成一个具体的对象

私有属性，在初始化函数中定义的属性，实例化时可以通过传参来定义自己的私有属性
