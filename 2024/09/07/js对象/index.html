BOM

browser Object Model， 浏览器对象模型，封装了浏览器的各个组成部分为对象

使得javascript与浏览器对话交互，

组成：

  • Window：浏览器窗口
  • Navigator：浏览器
  • Screen：屏幕
  • History：历史记录
  • Location：地址栏

window的属性和方法

执行window时，可以省略window.

属性：

  • history 调用history对象
  • location 调用location对象
  • navigator 调用navigator对象

方法：

  • alert() 警告框

  • comfirm() 对话框

  • setInterval() 周期性执行函数

  • setTimeout() 指定时间后执行函数

DOM

Document Object Model 文档对象模型，封装了标记语言的各个组成部分（

（html也是标记语言）

  1. Core DOM 所有文档的标准模型

组成：

  • Document 整个文档（html）
  • Element 元素对象（标签）
  • Attribute 属性对象 （标签中的属性）
  • Text 文本对象 （标签的value）
  • Comment 注释对象 （注释）

  1. XML DOM XML文档的标准模型

  2. HTML DOM HTML文档的标准模型

    html中的Element对象是由Document对象获取的，且Document是由window获取的，可以省略window.
