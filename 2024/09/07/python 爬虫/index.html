python 爬虫

爬虫基本流程

一、数据来源分析

  1. 明确需求： 明确采集的网站以及数据
    • 网址： https:// www.xxx
    • 数据：车次信息
    1. 抓包分析： 通过浏览器开发者工具分析数据位置
  • 网络模块查询
  • 找到(查票)接口网址

二、代码实现步骤

  1. 发送请求：模拟浏览器对于url地址发送请求
  2. 获取数据：获取服务器返回的响应数据
  3. 解析数据： 提取我们需要的数据内容
  4. 保存数据

数据获取

  • 此处的模拟浏览器仅是针对个别网站的反爬措施，不同网站根据具体情况破解反爬。
